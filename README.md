# Android MVVM + Clean Architecture

A sample Android application to show the MVVM design pattern with Clean Architecture.

## Features

- Kotlin
- MVVM design pattern
- Clean Architecture
- Koin
- Retrofit
- Moshi
- LiveData
- Coroutines
- Room
- Coil
- Navigation Component
- Mockk (for unit tests)
